# Practice Project

This is a simple Go practice project.

## Description

The program initializes a slice of integers from 0 to 10 and iterates through them to determine if each number is even or odd.

## How to Run

1. Make sure you have Go installed.
2. Run the program using the following command:

```bash
go run main.go
```

## Output

The program will output "Practice", followed by the slice of numbers, and then a line for each number indicating if it is EVEN or ODD.
